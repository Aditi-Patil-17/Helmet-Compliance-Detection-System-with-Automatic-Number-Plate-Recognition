Helmet Compliance Detection System with Automatic Number Plate Recognition (ANPR)
Overview

The Helmet Compliance Detection System is a computer vision-based traffic monitoring solution designed to automatically identify two-wheeler riders who are not wearing helmets and extract their vehicle registration numbers using Automatic Number Plate Recognition (ANPR).

The system processes live video streams in real time and assists traffic authorities by automating the detection and reporting of helmet safety violations, reducing the need for manual monitoring.

Objectives
Detect helmet compliance among two-wheeler riders.
Automate identification of traffic safety violations.
Extract vehicle registration numbers using OCR.
Enable digital challan generation for non-compliant riders.
Improve traffic monitoring efficiency through computer vision.

System Architecture
Camera captures live traffic video.
Video frames are processed using OpenCV.
Riders are detected in the scene.
Helmet compliance is verified.
Non-compliant riders are identified.
Vehicle number plates are localized.
OCR extracts registration numbers.
Violation records are generated for further action.
Implementation

The system was developed using computer vision and machine learning techniques to analyze live traffic feeds.

The implementation involved:

Video frame acquisition
Image preprocessing
Rider detection
Helmet classification
Number plate localization
OCR processing
Violation reporting

Machine learning models including Random Forest, KNN, HOG, and SVM were evaluated and integrated to improve detection accuracy and classification performance.

Results
Successfully detected helmet compliance from live traffic video streams.
Automated extraction of vehicle registration numbers using OCR.
Reduced dependency on manual traffic monitoring.
Demonstrated real-time violation detection and reporting capabilities.
Enabled automated digital challan workflow support.
